# Turtle with Your Mouse

通过tkinter分窗，简单实现了turtle跟随鼠标轨迹运动。

左窗口：鼠标操作窗口
右窗口：turtle跟随窗口

## 功能

* [x] 圆形
* [x] 矩形
* [x] 任意线条
* [x] 清屏

***

## 项目结构

```
├── README.md
├── main.py 主函数
├── init.py 窗口及turtle初始化
├── draw_func.py 绘制函数
├── utils.py
```
